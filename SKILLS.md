
# 🛠️ Project Skills

This repository contains custom skills for the Gemini CLI to automate repetitive workflows.

## 🗺️ `csv-mapper-creator` (User Scope)

**Description**: 
Creates a browser-based interactive map from a CSV dataset with drag-and-drop support and automatic coordinate detection. This skill encapsulates the design and publishing logic for standalone mapping tools.

### Key Functionalities:
- **Template-Based**: Uses a modern glassmorphism UI with Leaflet.js and PapaParse.
- **Privacy-First**: Processes all CSV data locally in the browser (no server uploads).
- **Automated Publishing**: Standardized workflow for deploying to GitHub Pages on a separate URL path via the `main` branch.

### How to use:
In a Gemini CLI session, you can trigger this skill by asking:
> *"Create a new CSV mapper for [X] data"* or *"Use the csv-mapper-creator to visualize [file.csv]"*
